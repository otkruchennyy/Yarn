ИНСТРУКЦИЯ ПО УСТАНОВКЕ / INSTALLATION GUIDE
=============================================

1. УСТАНОВКА ШРИФТОВ / FONT INSTALLATION
   - Переместите файлы шрифтов в папку "fonts"
   - Формат имени: "НазваниеШрифта.ttf" или "НазваниеШрифта.otf"
   - Для системных шрифтов укажите только название: "Arial"
	 / 
   - Move font files to the "fonts" folder
   - File name format: "FontName.ttf" or "FontName.otf"
   - For system fonts specify only the name: "Arial"

ПЕРЕЗАГРУЗИТЕ ПРИЛОЖЕНИЕ ПОСЛЕ УСТАНОВКИ
RESTART THE APPLICATION AFTER INSTALLATION
