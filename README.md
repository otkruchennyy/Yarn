# Yarn

Desktop приложение на PySide6.

## Установка
`git clone https://github.com/tsukagahara/Yarn.git`  
`cd Yarn`  
`python -m venv venv`  
`source venv/bin/activate`  # Linux/Mac  
`venv\Scripts\activate`     # Windows  
`pip install -r requirements.txt`

## Запуск
`python src/main.py`

## Технологии
Python 3.8+, PySide6
